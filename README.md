\# 🐍 Snake Water Gun Game



A simple command-line \*\*Snake Water Gun\*\* game built using \*\*Python\*\*. This project is based on the classic Snake-Water-Gun game, similar to Rock-Paper-Scissors.



\## 📌 Features



\- Play against the computer.

\- Random computer choice using Python's `random` module.

\- Detects:

&#x20; - ✅ Win

&#x20; - ❌ Lose

&#x20; - 🤝 Draw

\- Simple and beginner-friendly code.



\---



\## 🛠️ Technologies Used

\## 🎮 Game Rules



| Your Choice | Computer Choice | Result |

|-------------|-----------------|--------|

| Snake 🐍 | Water 💧 | Win |

| Water 💧 | Gun 🔫 | Win |

| Gun 🔫 | Snake 🐍 | Win |

| Same Choice | Same Choice | Draw |
